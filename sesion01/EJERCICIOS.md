# Sesión 01

## Concepto clave

- `print` -> imprime en la misma línea
- `println` -> imprime y hace salto de línea al final

## Ejercicios

### 1. Tu primer mensaje

Muestra tu nombre en pantalla.

### 2. Dos líneas separadas

Muestra exactamente:

```
Hola
Mundo
```

### 3. Todo en una línea

Muestra:

```
Hola Mundo
```

Usando más de un `print`

### 4. Mezcla inteligente

Muestra exactamente:

```
Hola Mundo
Soy [tu nombre]
```

Reemplazando `[tu nombre]` por tu nombre y en un `print` separado.
Es decir `System.out.println("Soy [tu nombre]")` es inválido.

### 5. Frase personalizada

Muestra:

```
Me llamo [tu nombre] y tengo [tu edad] años
```

De nuevo separando tu nombre y tu edad en `print`s diferentes.

### 6. Construyendo una frase

Muestra:

```
Java es divertido
```

Usando mínimo 3 `print`

### 7. Simular un ticket

Muestra algo como:

```
Producto: Pan
Precio: 2
Total: 2
```

Donde lo que va después de los dos puntos vaya en un `print` diferente.

### 8. Dibujar con texto

Muestra exactamente:

```
*****
*   *
*****
```

### 9. Repetición manual

Muestra tu nombre 3 veces en líneas distintas.

```
[tu nombre]
[tu nombre]
[tu nombre]
```

### 10. Desafío final

Reproduce EXACTAMENTE esto:

```
Hola, mi nombre es [tu nombre].
Estoy aprendiendo Java.
Y esto se ve en varias líneas.
Pero esta parte sigue en la misma línea -> listo.
```

Intenta no poner todo en el mismo `println` tratando de seccionarlo en varios `print`s independientes.

---

## Reglas

- Respeta espacios y saltos de línea
- No copies y pegues sin entender
- Prueba combinando `print` y `println`

## Consejo

Si no te sale un ejercicio no dudes en consultar.

## Meta

- Que puedas decidir cuándo usar `print` y cuándo usar `println` sin dudar.
- Lograr hacer mínimo 3 ejercicios de los propuestos.
