# Sesión 03

## Concepto clave

- Función -> realiza una acción
- Parámetros -> información que le damos a la función
- Retorno -> valor que la función devuelve
- Puedes usar el resultado de una función directamente o guardarlo en una variable

## Ejercicios

### 1. Usar una función

Imprime en consola un texto en mayúsculas usando una función.

Objetivo, mostrar:

```
HOLA
```

:: Usa un texto en minúsculas y transfórmalo.

### 2. Guardar el resultado

Transforma un texto a mayúsculas y guarda el resultado en una variable.
Luego muestra esa variable.

Objetivo:

```
HOLA
```

### 3. Dos transformaciones

Crea una variable con un texto.
Muestra ese texto en:

- Mayúsculas
- Minúsculas

Resultado esperado:

```
HOLA hola
```

### 4. Concatenación básica

Muestra tu nombre usando una variable y une el texto de saludo con tu nombre usando el método `String.concat(String string)`.

Resultado:

```
Hola [tu nombre]
```

:: Reemplaza `[tu nombre]` por tu nombre.

### 5. Concatenación

Crea una variable con tu nombre.
Muestra el siguiente mensaje usando una función:

```
Hola [TU NOMBRE]
```

:: Tu nombre debe estar en mayúsculas usando una función.

### 6. Potencias

Usa una función para elevar `3` a la `2` y muestra el resultado.

Resultado:

```
9
```

### 7. Guardar resultado numérico

Guarda el resultado de una potencia en una variable y luego muéstrala.

Resultado:

```
225
```

:: Usa base 15 y exponente 2.

### 8. Convertir texto a número

Convierte el texto `"10"` a número y súmale 5.

Resultado:

```
15
```

### 9. Detectar error

Este código no hace lo que esperamos:

```java
System.out.println("hola".toUpperCase);
```

Explica:
- ¿Cuál es el error?
- ¿Cómo lo solucionarías?

### 10. Desafío final

Crea una variable con tu nombre.

Reproduce EXACTAMENTE esto:

```
Hola [tu nombre] HOLA [TU NOMBRE] hola [tu nombre] Tu nombre tiene [número] letras
```

:: Usa funciones para transformar el texto 
 
:: `[número]` debe ser la cantidad de letras de tu nombre (busca en línea alguna función que te permita esto)
