# Sesión 02

## Concepto clave

- `int` -> números enteros
- `String` -> cadenas de texto
- `[tipo] [nombre] = [valor];`

## Ejercicios

### 1. Guardar y usar

Crea una variable, guarda tu nombre y luego usala para mostrar tu nombre 3 veces.

Objetivo, mostras lo siguiente:

```
[tu nombre]
[tu nombre]
[tu nombre]
```

:: Reemplaza `[tu nombre]` por tu nombre.

### 2. Frase con variable 1

Crea una variable y guarda tu nombre.
Usa esa variable para mostrar:

```
Hola, me llamo [tu nombre]
```

:: Reemplaza `[tu nombre]` por tu nombre.

### 3. Frase con variable 2

Crea una variable y guarda tu edad.
Usa esa variable para mostrar:

```
Hola, tengo [tu edad] años
```

:: Reemplaza `[tu edad]` por tu edad.

### 4. Dos variables

Crea dos variables, en una guarda tu nombre y en la otra, tu edad.
Usa esas variables para mostrar:

```
Hola, soy [tu nombre] tengo [tu edad] años
```

:: Reemplaza `[tu nombre]` por tu nombre.
:: Reemplaza `[tu edad]` por tu edad.

### 5. Cambiar valor

Crea una variable con el valor de `24` e imprimela en consola.
Luego reasignale `18` y vuelvela a imprimir.

Si está bien obtendrás:

```
24
18
```

### 6. Copiar variables

Crea una variable con tu nombre y muéstrala.
Luego crea una nueva variable y asígnale la variable que contenía tu nombre, también muestra esta nueva variable.

Si está bien obtendrás:

```
[tu nombre]
[tu nombre]
```

:: Reemplaza `[tu nombre]` por tu nombre.

### 7. Mini perfil

Crea tres variables, una con tu nombre, una con tu edad y una con tu color favorito.
Muestra algo como:

```
Hola, mi nombre es [tu nombre]
Mi edad es [tu edad]
Mi color favorito es [tu color]
```

:: Reemplaza `[tu nombre]` por tu nombre.
:: Reemplaza `[tu edad]` por tu edad.
:: Reemplaza `[tu color]` por tu color.

### 8. Detectar error

Este código tiene un error:

```
String nombre;
System.out.println(nombre);
```

Explica el porqué del error y como tu lo solucionarías para que muestre:

```
[tu nombre]
```

:: Reemplaza `[tu nombre]` por tu nombre.

### 9. ¿Es esto posible?

Crea una variable que tenga el número 4 y muestra su valor.
A esa variable reasignale el valor de si mismo más 3 y muestra su nuevo valor.

Si está bien deberías ver:

```
4
7
```

### 10. Desafío final

Crea tres variables, tu nombre, tu color favorito y el nombre de este lenguaje de programación (Java).

Reproduce EXACTAMENTE esto:

```
Hola, mi nombre es [tu nombre].
Estoy aprendiendo [el lenguaje].
Mi color favorito es [tu color].
Por cierto, mi nombre era [tu nombre].
Me confunde [el lenguaje].
```

:: Reemplaza `[tu nombre]` por tu nombre.
:: Reemplaza `[tu color]` por tu color.
:: Reemplaza `[el lenguaje]` por el lenguaje.

- No copies y pegues sin entender
- Siempre que te pidan crear una variable hazlo
