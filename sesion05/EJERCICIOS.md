# Sesión 05

## Concepto clave

- `boolean` -> true o false
- Operadores de comparación -> ==, !=, >, <, >=, <=
- Operadores lógicos -> && (AND), || (OR), ! (NOT)
- `if` -> ejecuta código si se cumple una condición
- `else` -> ejecuta código si la condición no se cumple

## Ejercicios

### 1. Mayor de edad

Crea una variable `edad` y verifica si es mayor o igual a 18.

Si lo es, muestra:

```
Eres mayor de edad
```

### 2. Mayor o menor

Crea una variable `edad` y usa `if-else` para mostrar:

```
Eres mayor de edad
```

o

```
Eres menor de edad
```

### 3. Número positivo

Crea una variable `numero`.

Si es mayor que 0 muestra:

```
El número es positivo
```

Si no:

```
El número no es positivo
```

### 4. Igual o diferente

Crea dos variables con números.

Si son iguales muestra:

```
Son iguales
```

Si no:

```
Son diferentes
```

### 5. AND (&&)

Crea dos variables:

- `edad`
- `tieneEntrada` (boolean)

Solo si cumple ambas condiciones muestra:

```
Puedes entrar
```

Si no:

```
No puedes entrar
```

### 6. OR (||)

Crea dos variables:

- `esAdmin`
- `esEditor`

Si al menos una es verdadera muestra:

```
Tienes acceso
```

Si no:

```
Acceso denegado
```

### 7. NOT (!)

Crea una variable `estaLloviendo`.

Si NO está lloviendo muestra:

```
Podemos salir
```

### 8. Con Scanner

Pide al usuario su edad.

Si es mayor o igual a 18 muestra:

```
Puedes entrar
```

Si no:

```
No puedes entrar
```

### 9. Dos condiciones con Scanner

Pide al usuario:

- Su edad
- Si tiene entrada (`true/false`)

Muestra:

``
Bienvenido
```

solo si cumple ambas condiciones.

Si no:

```
Acceso denegado
```

### 10. Desafío final

Pide al usuario:

- Su nombre
- Su edad

Y muestra EXACTAMENTE esto:

```
Hola [nombre] Eres mayor de edad
```

o

```
Hola [nombre] Eres menor de edad
```

:: Usa `if-else`  
:: Reemplaza `[nombre]` por el valor ingresado

## Reglas

- Respeta espacios y saltos de línea
- No copies y pegues sin entender
- Usa correctamente `if` y `else`
- Usa operadores cuando sea necesario

## Consejo

Antes de escribir código, intenta responder la condición en tu cabeza:
¿qué tiene que pasar para que sea `true`?

## Meta

- Usar correctamente operadores de comparación
- Combinar condiciones con lógica
- Resolver mínimo 3 ejercicios
